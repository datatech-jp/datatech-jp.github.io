# datatech-jp.github.io

## 概要
データエンジニアリング、データ活用についての知見を共有したり、交流するためのコミュニティです。

## 参加条件
年齢・所属問わず、データを使ってなにかしたい、データエンジニアリングやデータ活用に興味がある方ならだれでも。
